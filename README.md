# todo-list-js
ToDoList를 웹페이지로 만들었습니다.
주요 기능으로는 텍스트를 추가, 수정, 삭제 및 완료 버튼이 있습니다.
텍스트를 입력후 따로 버튼을 누를 필요 없이 'Enter'키를 눌러 편하게 내용을 입력 할 수 있습니다.
핵심 기능
  1. DOM 조작을 활용해 새로고침을 사용하지 않고 지속적으로 화면을 수정했습니다.
  2. LocalStorage & JSON을 활용하여 객체를 문자열로 변경후 LocalStorage에 저장하고 불러오는 기능을 추가했습니다
  3. index를 활용해 상호작용 하고 싶은 텍스트에 편하게 접근할 수 있었습니다
